深度搜索问题的代码模板

  1. 对于if的判定条件，如果判定过程比较复杂的话，要封装成isOk()k函数，一般来说深度搜索的问题，关键点在于isOk函数的创建。
  2. 矩阵的深搜问题，边界判断条件一定是不能忘的， > x>=0 && x=0 && y<n
  3. 考虑是否要设立标数组，并且，在合适的情况下是否要进行回溯。
  4. 当在多选择深度搜索的时候，一个for循环里面嵌套递归函数，在深度搜索寻找到一个合适的递归函数返回值之后，考虑是否需要直接将该正确结果返回，并结束该循环。

矩阵深搜的模板牢记于心:

  • 外层循环
  • 得到新的状态
  • 进行判断当前的状态是否合法
  • 如果合法进行递归深搜

全排列问题

问题定义：求一个集合的所有全排列

方法一： 使用for循环深度搜索

class Solution {
public:
    vector<vector<int>> res;
    vector<vector<int>> permute(vector<int>& nums) {
        vector<int> ans;
        vector<int> vis(nums.size(), 0);
        dfs(ans, vis, nums);
        return res; 
    }
    void dfs(vector<int>& ans, vector<int>& vis, vector<int> nums){
        if(ans.size()==nums.size()){
            res.push_back(ans);
        }
        for(int i=0;i<nums.size();i++){
            if(!vis[i]){
                ans.push_back(nums[i]);
                vis[i] = 1;
                dfs(ans, vis, nums);
                ans.pop_back();
                vis[i] = 0;
            }
        }
    }
};

方法二：使用交换思想深度搜索

class Solution {
public:
    vector<vector<int>> res;
    vector<vector<int>> permute(vector<int>& nums) {
        vector<int> vis(nums.size(), 0);
        dfs(nums, 0);
        return res; 
    }
    //k表示第几层
    void dfs(vector<int>& nums, int k){
        if(k==nums.size()-1){
            res.push_back(nums);
            return;
        }
        int temp = 0;
        for(int i=k;i<nums.size();i++){
            //交换
            temp = nums[k];
            nums[k] = nums[i];
            nums[i] = temp;
            dfs(nums, k+1);
            //再次交换回来
            temp = nums[k];
            nums[k] = nums[i];
            nums[i] = temp;
        }       
    }
};

小结

  • 递归树的构建
  • 对于for循环式深度搜索，该递归函数自带有递归出口，即当循环执行完之后结束
  • 标记数组的必要使用
  • 方法一的结果集合是针对原集合的元素顺序得出的全排列结果集合
